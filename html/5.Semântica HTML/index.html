Visualize a estrutura html no código

Imagem de referência para semãntica do html

< header > Usado para por o cabeçalho como logo e menus de navegação

< nav > Usado para por os links de navegação utilizando geralmente outras tag como < ul > e < li >

< main > Define o conteúdo principal

< article > usado para marcar uma parte independe e autocontida do conteúdo como notícias

< section > usado para organizar em seções

< aside > é geralmente usado para conteúdo relacionado, mas não central, ao conteúdo principal da página

< footer > é geralmente usado para adicionar o rodapé como link para redes sociais ou contatos
