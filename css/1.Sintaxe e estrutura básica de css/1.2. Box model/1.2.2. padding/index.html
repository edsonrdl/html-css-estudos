Padding

A parte cinza faz referência ao elemento inteiro,só é possível visualizar a parte cinza por conta do padding que cria esse espaçamento entre a borda da caixa cinza com a borda da caixa amarela (conteúdo)

Forma de usar o padding

padding:50px ou padding: 50px(Para cima) 50px(Para direita) 50px(Para baixo) 50px(Para esquerda)

Padding

Outras formas de usar o padding

Espaçamento pada cima

padding-top: 50px;

Espaçamento pada baixo

padding-bottom: 50px;

Espaçamento pada esquerda

padding-left: 50px;

Espaçamento pada direita

padding-right: 50px;
